# 工具介绍

## 消费清单导出工具

您登录(您只需要手动输入验证码)之后可以自动爬取您的消费清单，并且导出为csv文件。

可以用Excel/WPS直接打开，或者可以期待一下我的分析工具！

### 安装要求

```
selenium>=4.0.0
webdriver_manager>=4.0.0
```

程序需要 selenium 分析网页，
需要webdriver_manager自动获取webdriver！

### 使用说明

1. 安装依赖
2. 请修改`template/shmtu_num_pwd.numpwd`中内容。
3. 运行`main.py`，输入验证码，等待程序自动运行
